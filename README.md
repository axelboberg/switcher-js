# Switcher JS
A video switcher implemented in Javascript for running in Google Chrome

## Usage

### Setup
Clone the repo and open [index.html](index.html) in Google Chrome.

### Keyboard shortcuts
| Key | Action |
| --- | --- |
| 1-8 | Set the preview to input 1-8 |
| Space | Cut |
